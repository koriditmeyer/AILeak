Features

Leveraging AI, IoT and cloud technologies, we've created LeakAI Cloud and LeakAI Detector to streamline the time-consuming, costly process of detecting leaks in potable water networks.

LeakAI Cloud is a secure SaaS solution, accessible anywhere and capable of offline data recording. It records real-time field device data, processes it through machine learning algorithms to identify noise patterns, and enables acoustic correlation orders for precise leak detection.

LeakAI Detector is an innovative field device, combining acoustic recording and correlation in a robust, low-cost microcontroller. Together, these tools automate the entire leak detection process, enhancing accuracy and efficiency.

Fast and Accurate Leak Detection

LeakAI offers a fast, accurate, and intelligent solution to water leak detection. Using IoT and AI, it combines LeakAI Cloud and LeakAI Detector for real-time, cloud-based analysis.

The in-house developed software allows for off-line data recording and updates when internet connection is re-established. Advanced machine-learning algorithms analyse noise data to locate leaks, while acoustic correlation orders can be sent for further analysis.

Paired with any compatible noise recording device, the LeakAI system revolutionizes leak detection, enhancing efficiency and precision.

Powered by IoT, spatial and cloud data management, and AI, they allow real-time data collection and analysis. LeakAI Cloud is accessible from any internet-connected device, and it can record data offline, updating when connectivity is restored.

How it works

  1. LeakAI is a solution for detecting leaks in potable water networks, significantly improving the traditionally time-consuming and complicated process by using connected devices, cloud-based spatial data management, and artificial intelligence.
  2. The solution is made up of LeakAI Cloud and LeakAI Detector, where LeakAI Cloud is a Software-as-a-Service (SaaS) offering based on a scalable spatial data management solution developed in-house.
  3. LeakAI Cloud can be accessed from any internet-connected device, provides a dedicated web application for enhanced usability, and allows data recording offline with updates once the internet connection is restored.
  4. The system records real-time field device data in the cloud, allowing analysis with machine learning-based algorithms that learn from collected noise data and detect trends, day or night, while effectively eliminating daytime noise.
  5. It also facilitates acoustic correlation orders to field devices, which use GIS data (pipe length between two devices, material, diameter) to analyze noise spikes and identify leak locations.
  6. For better leak pinpointing, LeakAI Cloud utilizes Aquaphones for ground listening at the location suggested by the correlation. These devices, equipped with a sound amplifier, increase perceived sounds tenfold and can localize leaks without prior knowledge thanks to their connection with LeakAI Cloud.
  7. LeakAI Cloud allows agents to log found leaks directly from their mobile devices for traceability.
